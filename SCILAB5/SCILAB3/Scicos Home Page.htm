Scicos: a dynamic systems modeler and simulator

Scicos is a Scilab toolbox included in the Scilab Package.  Written in Fortran, C and Scilab language, it comes with complete source code. Scicos is free software.

Scicos features are:

  • A user-friendly GUI-based editor for modeling dynamical systems as block diagrams
  • Hierarchical block diagram structure (Super blocks)
  • A large number of standard blocks available in various palettes
  • Possibility of defining new blocks using C, Fortran (dynamic link) or Scilab Language
  • Powerful formalism for modeling hybrid systems
  • Diagram compilation and fast simulation

What is the latest version?

There are no specific version numbers associated with Scicos releases. Since each release of Scicos corresponds to a new Scilab release, we refer to each Scicos release by the version number of the associated Scilab release.
The latest version is Scicos 2.6

Does it run on my computer?

If you have Scilab-2.6 up and running on your system, then you have Scicos. Scicos is a built-in toolbox of Scilab.

Scilab is free software and works on Unix-Xwindow systems (including Linux)and Windows 95, 98 and NT.
download Scilab

Does it work?

 Yes it does! The underlying hybrid system formalism is powerful enough for modeling many complex systems.
  • Scicos has been successfully used in a number of large industrial applications.
  • Scicos is used for teaching systems, control and signal processing.
  • But Scicos is not bug free! And there is much to be done.

What is new?

Beta versions of Scicos have been distributed with earlier versions of Scilab. Scicos 2.4.1 is the first stable release of Scicos. The new features compared to earlier versions are too numerous to list here.

The followings are what has been done since the release 2.4:

  • many bugs have been fixed (and surely many new ones introduced)
  • conditional branching now works for continuous time signals too
  • on-line block construction using incremental link function now works on all platforms
  • new features added to the diagram editor

What is going on?

The development of Scicos continues. So far it has been done exclusively at  INRIA but we are hoping for outside contributions. On-going projects are:
  • Real-time code generation via  SynDEx  interface
  • Design of more sophisticated scopes, including a floating scope
  • Implicit blocks (differential-algebraic constraints) and use of dassl for simulation
  • Facilities for on-line, block and super-block, documentation

Interested in contributing?

The simplest way to contribute is to report bugs to Dr Scilab. But if you want to contribute to the development of Scicos, then we suggest you prepare a project plan and post it to the scilab newsgroup comp.soft-sys.math.scilab. This way you can get the reaction of the Scicos community to your project, and you may even find assistance for realizing it.

Type of projects we are thinking of are for example:

  • Signal processing toolbox (set of specialized basic blocks devoted to signal processing applications)
  • Super block compilation (possibility of separately compiling subdiagrams  of a Scicos diagram)
  • Device read and write (e.g. read from a microphone, process, write to speaker, in real-time)
  • New data types (currently Scicos signals are coded only as double precision arrays)
